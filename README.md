## Japanese crosswords solver
by Vyacheslav Dubinin <vyacheslav.dubinin@gmail.com>
  * http://code.google.com/p/jc-solver/
  * https://habrahabr.ru/post/151819/

build
```
  cabal build
```
run
```
  ./jc-solver -i example -o example.pmb
```
run with interpretator
```
  cd src
  ghci
  Prelude> :l test-01
  Prelude> t1
```
